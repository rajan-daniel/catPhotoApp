## 🐱 Cat Photo App (HTML Project)

### 📌 Preview  
This project is a simple “Cat Photo App” built using basic HTML. It demonstrates how different HTML elements come together to create a structured and functional webpage.

---

### 📖 About the Project  
HTML tags define the structure of a webpage and organize its content. In this project—my first experience working with HTML—I built a Cat Photo App while learning and applying the most commonly used tags.

---

### 🚀 What I Learned  
- How to use **headings** and **paragraphs** to structure content  
- Adding **images** and **links** to a webpage  
- Creating **lists** (ordered and unordered)  
- Using **buttons** and basic **form elements** for user input  
- Understanding how HTML provides the foundation of web development  

---

### 🛠️ Technologies Used  
- HTML5  

---

### 🎯 Key Takeaway  
This project gave me a strong foundation in HTML and helped me understand how to build and structure a basic webpage from scratch.
